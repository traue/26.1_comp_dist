🔧 Como Funciona?

1

socket()

O servidor cria um socket TCP — um ponto de comunicação na rede.

→
2

bind()

Associa o socket a um endereço IP e porta (ex: 0.0.0.0:8080).

→
3

listen()

Coloca o socket em modo escuta, pronto para receber conexões.

→
4

accept()

Aguarda e aceita conexões de clientes (seu navegador!).

→
5

read() / write()

Lê a requisição HTTP e envia a resposta com o arquivo solicitado.
